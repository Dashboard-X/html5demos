JavaScript Redirection Script

JavaScript redirect is a client side redirecting technique. This tool helps you to automatically create redirect scripts momentaneously. You can also add time delay before the redirection, or you can just set it to zero, and create an immediate redirection. If you can't access the server's .htaccess file, or have no PHP code to work with, this tool will be your best friend to manage redirections on your websites. Here are some situations, when you'll be in need of JavaScript redirection:

  • redirect blogger and blogspot
  • redirect wordpress
  • redirect tumblr
  • redirect html website

The redirect script uses the window.location.replace(url) instead of the window.location.href = url to prevent the browser to push the redirector site to the history, so the users are able to hit the back button, without creating redirect loops. The script also contains an html redirect code line only for JavaScript disabled browsers. It also has a built-in IE hack to pass the http referer. A link rel="canonical" line is added to the script, so search engines will display your destination site instead of the redirector.
